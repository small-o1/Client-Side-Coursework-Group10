## 📖 About

We are a non-profit library aiming to help kids learn. Our website is a completely **frontend, client-side multi-page** web application built with **semantic HTML**, **CSS**, **JavaScript**, and **jQuery**.

### ✨ Features

- **📚 Request Books** — Users can browse and submit book requests.
- **📋 View Requests** — See a list of all previously submitted requests.
- **✅ Form Validation** — Client-side validation on all input fields.
- **⚡ Dynamic Event Handling** — Interactive UI built with jQuery event handling.
- **💾 Client-Side Storage** — All data is persisted using **localStorage** (or sessionStorage where appropriate).

### 🛠 Tech Stack

| Technology      | Purpose                             |
|-----------------|-------------------------------------|
| Semantic HTML5  | Accessible, meaningful markup       |
| CSS3            | Styling and layout                  |
| JavaScript      | Core application logic              |
| jQuery          | DOM manipulation & event handling   |
| localStorage    | Client-side data persistence        |
